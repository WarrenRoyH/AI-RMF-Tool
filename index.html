The Manage phase (Sentry) supports three distinct deployment architectures to fit your existing application stack:

1. Library-Based Directly import core/sentry.py into your Python app for zero-latency safety checks.
2. Middleware Proxy Run a standalone "Secure Firewall" that intercepts all traffic before it reaches the model.
3. Prompt-Based Inject manifest guardrails directly into the System Prompt for lightweight enforcement.
